# Old Fashioned Media Board

A simple black-and-white static website with an old-fashioned look.

## Features

- Black background with white text and vintage styling
- Upload images and videos from your browser
- Add optional descriptions to posts
- Add links with optional link text
- Posts are saved locally in the browser via `localStorage`

## Usage

1. Open `index.html` in a browser.
2. Use the form to choose a media file, add a description, or post a link.
3. Submitted posts will appear in the right-hand panel.

## Notes

- This is a frontend-only demo. A backend will be needed later for real file uploads and persistent storage across devices.
- The site is intentionally styled to feel old-fashioned and will be easy to update as the design evolves.
